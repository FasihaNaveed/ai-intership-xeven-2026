# Day 3 Learnings

## Key Concepts
- Conditional statements: if, elif, else for decision making
- Boolean logic: True, False, truthy and falsy values
- Comparison operators: ==, !=, >, <, >=, <=
- Flow control: program execution depends on conditions
- Nested if statements for complex decision making
- Input validation using condition-based error handling
- Common mistakes: = vs == and indentation errors
