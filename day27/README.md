# 🚀 Day 27 - Production RAG System

## 📌 Overview
This project is a production-ready RAG backend built with FastAPI, SQLAlchemy, and Docker.

---

## ⚙️ Features

- FastAPI REST API
- SQLite Database (Upgradeable to PostgreSQL)
- Document Management System
- Conversation Tracking
- Analytics Endpoint
- Export Data API
- Rate Limiting Middleware
- Docker Support
- Unit Testing (pytest)
