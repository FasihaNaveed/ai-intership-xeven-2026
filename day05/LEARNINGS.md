# Day 5 Learnings

## Key Concepts

* Machine Learning (ML) enables systems to learn patterns from data and make predictions or decisions.
* Supervised Learning uses labeled data (input-output pairs) to train models.
* Unsupervised Learning discovers hidden patterns in unlabeled data.
* Reinforcement Learning learns through trial and error using rewards and penalties.
* Regression predicts continuous numerical values.
* Classification predicts discrete categories or labels.
* Decision Trees make decisions using branching conditions similar to if-else statements.
* Model performance depends on data quality, training process, and decision logic.

## Best Practice Learned

Before selecting a Machine Learning approach, first identify:

* Whether data is labeled or unlabeled
* Whether output is continuous or categorical
* Whether the system should learn through rewards

Choose algorithms based on the actual problem instead of applying one model everywhere.
