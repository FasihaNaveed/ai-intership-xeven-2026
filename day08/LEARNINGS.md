# Day 8 Learnings

## Key Concepts

* Python lists are ordered and mutable collections that can store multiple data types.
* List methods help add, remove, update, and organize data efficiently.
* List slicing allows extraction of specific portions of a list using start, end, and step values.
* Negative indexing provides access to elements from the end of a list.
* List comprehensions provide a concise and readable way to create and filter lists.
* Lists are commonly used for managing collections such as students, products, records, and datasets.
