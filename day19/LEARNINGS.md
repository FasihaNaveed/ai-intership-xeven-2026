# Day 19 – Prompt Engineering Mastery

## Key Learnings

### 1. Zero-shot Prompting
- No examples required
- Works for simple tasks

### 2. Few-shot Prompting
- Uses examples
- Improves accuracy significantly

### 3. Chain-of-Thought (CoT)
- Encourages step-by-step reasoning
- Better for complex tasks

### 4. System Messages
- Define role, behavior, constraints

### 5. Prompt Patterns
- Persona
- Instruction
- Context
- Format
- Constraints

### 6. Common Mistakes
- Vague instructions
- No output format
- Conflicting rules
